### My SQL Query
```sql
WITH TweetLengths AS (
  SELECT t.user_id, LENGTH(t.text) AS text_length
  FROM tweets t
)
SELECT u.user_name, AVG(tl.text_length) AS avg_text_length
FROM TweetLengths tl
JOIN users u ON tl.user_id = u.user_id
GROUP BY u.user_name
ORDER BY avg_text_length DESC;
```

#### Top Users by Tweet Count (CTE)
```sql
WITH UserStats AS (SELECT user_id, COUNT(*) as tweet_count FROM tweets GROUP BY user_id) SELECT user_id, tweet_count FROM UserStats ORDER BY tweet_count DESC LIMIT 10
```
